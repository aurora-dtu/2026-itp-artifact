    Images and preimages of sets #

    Main definitions #

    • preimage f t : Set : the preimage f(t) (written f ' t in Lean) of a subset of .

    • range f : Set : the image of univ under f. Also works for {p : Prop} (f : p ) (unlike image)

    Notation #

    • f ' t for Set.preimage f t

    • f '' s for Set.image f s
