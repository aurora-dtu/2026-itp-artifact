    Filter.atTop and Filter.atBot for intervals in a linear order topology #

    Let X be a linear order with order topology. Let a be a point that is either the bottom element of X or is not isolated on the left, see Order.IsSuccPrelimit. Then the Filter.atTop filter on Set.Iio a and ùìù[<] a are related by the coercion map via pushforward and pullback, see map_coe_Iio_atTop and comap_coe_Iio_nhdsLT.

    We prove several versions of this statement for Set.Iio, Set.Ioi, and Set.Ioo, as well as Filter.atTop and Filter.atBot.

    The assumption on a is automatically satisfied for densely ordered types, see Order.IsSuccPrelimit.of_dense.
